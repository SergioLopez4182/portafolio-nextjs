# 🚀 Personal Portfolio – Projects Showcase

This repository contains the source code of my personal portfolio website, where I showcase selected professional projects I’ve worked on as a Full Stack Developer.

⚠️ **Note:** The projects presented here do not include their original source code, as it belongs to the companies I worked for. This portfolio focuses on describing the architecture, challenges, responsibilities, and results of each project. 
I will progressively add independent demo implementations where possible.

## 🌐 Live Website

https://portfolio-sergio-ls-pied.vercel.app

## ✨ About This Project

This portfolio was built using **Next.js** to simplify deployment on Vercel and to leverage modern React features and routing capabilities.

The goal was to create a clean, structured, and scalable showcase platform with:

- 🌍 Internationalization (English / Spanish)
- 🌙 Dark mode support
- 📱 Fully responsive layout
- 🧱 Modular project structure
- 🗂 Custom project listing system
- 🎨 Combination of Mantine components with Tailwind utilities
- ⚡ Optimized for deployment on Vercel
